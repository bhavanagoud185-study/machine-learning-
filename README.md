# XGBoost in Practice: Investigating Tree Depth and Learning Rate

## Overview

This tutorial demonstrates how hyperparameters influence the performance of XGBoost models. The analysis investigates the effect of tree depth and learning rate on classification performance using a Kaggle heart disease dataset.

The tutorial was created as part of a Machine Learning tutorial assignment.

## Dataset

Heart Disease Prediction Dataset (Kaggle)

The dataset contains medical attributes used to predict whether a patient has heart disease.

Target variable:
HeartDisease

Dataset source:
https://www.kaggle.com/datasets/fedesoriano/heart-failure-prediction

## Tutorial Content

The notebook demonstrates:

- Data preprocessing
- Baseline XGBoost model
- Effect of tree depth on performance
- Effect of learning rate on performance
- Training curve analysis
- Hyperparameter interaction heatmap
- Model evaluation
